* Використовуйте bootstrap i material ui
* Роути users, posts, comments повинні завантажуватись через lazy loading (ліниве завантаження)
* Дані users отримувати через fetch
* Дані posts, comments отримувати через axios
* Старайтесь все максимально оптимізувати, перевикористовувати компоненти

1. В роутах users, posts, comments повинні бути списки відповідних даних з https://jsonplaceholder.typicode.com/

2. На кліку на кожний елемент списку повинна відкриватись нова сторінка з конкретним елементом

3. Через query params напишіть запит і компоненту FilterUser і 2 поля вводу (для id та name юзера),
   коли клікаємо на кнопку потрібно показати результат пошуку.

4. Створіть компоненту Practice всередині якої ви будете створювати нові сторінки та робити
   запити (через axios) на https://jsonplaceholder.typicode.com/
   (Чим більше прикладів з лекції, документації - тим краще)
